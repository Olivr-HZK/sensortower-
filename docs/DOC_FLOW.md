## 📋 详细步骤说明

### 步骤 1：检查配置
```javascript
var docId = getConfig(CONFIG_KEYS.GOOGLE_DOC_ID, null);
```
- 从 PropertiesService 读取配置的 Doc ID
- 如果未配置，返回 `null`

### 步骤 2：尝试打开现有文档
```javascript
if (docId) {
  try {
    doc = DocumentApp.openById(docId);
  } catch (e) {
    doc = null; // 打开失败
  }
}
```
- 如果配置了 Doc ID，尝试打开
- 成功：使用现有文档
- 失败：需要创建新文档

### 步骤 3：创建新文档
```javascript
if (!doc) {
  var docName = "📊 周报汇总归档 - " + ss.getName() + " - " + formatDate(new Date());
  doc = DocumentApp.create(docName);
  docId = doc.getId();
  setConfig(CONFIG_KEYS.GOOGLE_DOC_ID, docId);
}
```
- 使用 `DocumentApp.create()` 创建文档
- 文档名称包含表格名称和日期
- 自动保存 Doc ID 到配置

### 步骤 4：设置初始内容
```javascript
var body = doc.getBody();
body.clear();
body.appendParagraph("📊 市场趋势监测周报归档")
    .setHeading(DocumentApp.ParagraphHeading.TITLE);
body.appendParagraph("")
    .appendText("本文档用于自动归档每周的市场监测周报。")
    .setFontSize(10)
    .setForegroundColor("#666666");
body.appendHorizontalRule();
```
- 清空默认内容
- 添加标题和说明
- 添加分隔线

### 步骤 5：追加周报内容
- 从「📊 周报汇总」表读取数据
- 格式化并追加到文档
- 包含表格和链接

### 步骤 6：保存并提示
```javascript
doc.saveAndClose();
// 显示成功消息和文档链接
```
- 保存文档
- 显示文档链接
- 提示用户访问

## 🎯 关键 API

### DocumentApp.create(name)
- **功能**：创建新的 Google Doc
- **参数**：文档名称（字符串）
- **返回**：Document 对象
- **位置**：Google Drive 根目录

### DocumentApp.openById(id)
- **功能**：通过 ID 打开现有文档
- **参数**：文档 ID（字符串）
- **返回**：Document 对象
- **异常**：如果文档不存在或无权限，抛出异常

### PropertiesService.getDocumentProperties()
- **功能**：存储和读取配置
- **存储位置**：与 Google Sheets 绑定
- **权限**：只有拥有编辑权限的用户可访问

## 💡 最佳实践

1. **首次使用**：不配置 Doc ID，让系统自动创建
2. **文档管理**：使用一个文档归档所有周报
3. **定期备份**：重要数据定期导出备份
4. **文档整理**：创建后可以移动到指定文件夹
